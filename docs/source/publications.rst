Publications using aPriori
==========================

This page collects scientific works that use or reference **aPriori** /
**aPrioriDNS**.  
If you used the library in a publication and it is not listed here yet,
please get in touch so we can add it.

.. note::

   Replace the example entries below with your actual publications
   (or extend the lists). You can group them however you prefer.

How to cite aPriori
-------------------

If you use aPriori in your work, please include a citation similar to:

.. code-block:: text

   L. Piu, *aPrioriDNS: a Python library for processing DNS data of reacting
   and non-reacting flows*, version X.Y.Z, YEAR.
   Available at: https://github.com/LorenzoPiu/aPrioriDNS

(Adjust the author list, title, version and year as appropriate.)
